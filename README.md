# 🏢 Khadija Real Estate Platform

A modern Real Estate Management Platform developed using Core PHP and MySQL. The platform allows users to explore properties, view detailed information, send inquiries, and enables administrators to manage listings efficiently.

## 🚀 Features

- Responsive Real Estate Website
- Property Listings and Details
- Property Search and Filtering
- Inquiry and Contact Forms
- Admin Dashboard
- CRUD Operations for Properties
- Database-Driven Application
- Mobile-Friendly UI

## 🛠 Tech Stack

- Core PHP
- MySQL
- HTML5
- CSS3
- JavaScript
- Bootstrap 5

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/build-with-zahir/khadija-real-estate-platform.git
```

2. Import the database

```bash
setup.sql
```

3. Configure database connection.

4. Run the project on:

```
http://localhost/khadija_real_estate/
```
